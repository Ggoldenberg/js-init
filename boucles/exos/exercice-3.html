Javascript - Boucle exercice 3

Réglage automatique de thermostat

Compléter le code situé dans le fichier js/exercice-2.js afin de :

Créer un système qui permettrait à un thermostat de régler sa température automatiquement jusqu'à atteindre une valeur cible

  • Dans une première version vous ne prendrez en charge que le cas ou la température initiale est inférieur à la température cible
  • Dans une deuxième version vous pourrez ajouter le cas ou la température initiale est supérieure à la température cible
