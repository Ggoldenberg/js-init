Verificateur de mot de passe

Compléter le code situé dans le fichier js/exercice-2.js afin de :

Ecrivez un script permettant de dire si le mot de passe fourni peut être considéré comme "fort"

Un mot de passe sera considéré comme fort si

  • Sa longueur est supérieure à 8 caractères
  • Il contient un "@" ou un "_"
  • Il contient un "T" en 4 ème position
