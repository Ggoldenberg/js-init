Javascript - Condition exercice 3

Système de détermination de prix

Compléter le code situé dans le fichier js/exercice-3.js afin de :

Créer un système qui determine le prix d'une article en fonction du type de client.

Un client peut être de 3 types différents

  • Particulier -> le prix de l'article sera alors de 100 euros
  • Professionnel -> le prix de l'article sera alors de 50 euros
  • Comme faisant partie du personnel -> le prix de l'article sera alors de 25 euros
  • Si aucun des cas cités, alors un message devra indiquer que l'on en peut pas determiner un prix car le type de client renseigné est invalide
