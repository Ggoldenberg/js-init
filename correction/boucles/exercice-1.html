Javascript - Boucle exercice 1

Décompte

Compléter le code situé dans le fichier js/exercice-1.js afin de :

Réaliser un décompte d'apres un nombre renseigné par l'utilisateur Pour cet exercice on demandera à l'utilisateur (à l'aide de prompt) un nombre à l'utilisateur entre 0 et 10, on affichera ensuite tous les nombres sous le nombre indiqué par l'utiliseur (si il rentre 4, on afficher 4, 3, 2, 1, 0). Si l'utilisateur rentre un chiffre erroné on affichera un message d'erreur.
